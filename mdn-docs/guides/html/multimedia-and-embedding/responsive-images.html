Responsive Images

In this article we will learn about the responsive image. If the images works well in mobile, tablet and desktop then we can call it responsive. This also helps to improve the perfomance.

Why responsive images?

The image in desktop version is bigger than the image in mobile or tablet due to difference in screen sizes. So the image looses it details when it is appear in mobile or tablet. Some desktop uses high resolution screens that need larger images to display nicely.

So the responsive images technique helps to improve the performance and UI image responsive.

How do you create responsive images?

We can provide the srcset and sizes attritube to the image to tell browser to pick the right one.
