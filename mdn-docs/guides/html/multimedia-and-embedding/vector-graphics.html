Vector Graphics

There are two types of images, raster images and vector images.

  • Raster images are shows exactly where the pixel is placed and what color it is. Such as png, jpeg, gif etc.
  • Vector images defined using algorithm, a vector image contains the shape and defination that computer can use to work out what the image should look like.

We can use SVG and PNG

The first one is svg and second one is png. They are identical but they are different. When zoom in, the png images gets started to pixelate.

What is SVG ?

SVG is an XML-based language for describing vector images. SVG is for marking up graphics not contents. SVG defines elements for creating basic shapes, like <circle> and <rect> as well <path> and <ploygon> for complex shapes.

SVG has some additional advantages besides those described so far:

  • Text in vector images remains accessible (which also benefits your SEO).
  • SVGs lend themselves well to styling/scripting, because each component of the image is an element that can be styled via CSS or scripted via JavaScript.

So why would anyone want to use raster graphics over SVG? Well, SVG does have some disadvantages:

  • Complex svg can take significant time in the browser.
  • SVG is harder to create ratser image.

Adding SVG to your pages

Here we will learn about adding svg in the web page.

<img> element

To embed the svg in the web page, we can use the <img> element with src pointing to the svg path.

We can also use svg as background image using css.

How to include SVG code inside your HTML

We can also write svg directly in the HTML code. This is called inline svg.
