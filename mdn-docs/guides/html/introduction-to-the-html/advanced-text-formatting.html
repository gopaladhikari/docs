    Representing computer code

    There are multiple ways to represent computer code using HTML.

    1. <code>
    For marking up generic pieces of computer code.
    2. <pre>
    For marking up preformatted text, means whitespaces will be rendered identically.
    3. <var>
    For specifically marking up variable names.
    4. <kbd>
    For marking up keyboard input entered into the computer.
    4. <samp>
    For marking up the output of a computer program
