    • Hyperlinks

    Hyperlinks allows us to link document to documentm or resources.

    A URL link can point to HTML file, text file, images, audio video.

    Anatomy of a link

    A basic hyperlink is created using <a> tag, passing the url to the href attritbute.

    I'm creating a link to the Mozilla homepage.

    Block level links

    Any content can be converted into block level link by wrapping block level element within <a> tag.

    Image links

    Image can be also converted into links by wrapping image element within <a> tag.

    We can also pass title attritbute to the <a> tag which shows up when hovered. It only rely on mouse hover not in screen reader, touchscreens etc.

    Document fragments

    Specific part of HTML document is known as document fragments and it is possible to link the specific part of document with the help of id.

    Absolute versus relative URLs

    Absolute URLs always points to exactly same location on web.

    Relative URLs always points to location relative to the current page.
