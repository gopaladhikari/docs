    Lists

    List are everywhere for example, shopping list,todo list, direction to house etc.

    There are three types of three types of lists: unordered, ordered, and description.

    Unordered

    Unordered lists are used to mark up lists of items for which the order of the items doesn't matter. Let's take a shopping list as an example:

    Unordered list starts with <ul> which warps around list items i.e <li>.

    • milk
    • eggs
    • bread
    • hummus

    Ordered

    Ordered lists are lists in which the order of the items does matter. Let's take a set of directions as an example

    Ordered list starts with <ul> which warps around list items i.e <li>.

    1. Drive to the end of the road
    2. Turn right
    3. Go straight across the first two roundabouts
    4. Turn left at the third roundabout
    5. The school is on your right, 300 meters up the road

    Nesting list

    Its okay to nest one list inside another.Let's take the second list from our recipe example:

    1. Remove the skin from the garlic, and chop coarsely.
    2. Remove all the seeds and stalk from the pepper, and chop coarsely.
    3. Add all the ingredients into a food processor.
    4. Process all the ingredients into a paste.
    5. If you want a coarse "chunky" hummus, process it for a short time.
    6. If you want a smooth hummus, process it for a longer time.

    Since the last two bullets are very closely related to the one before them,it might make sense to nest them inside their own unordered list and put that list inside the current fourth bullet. This would look like so:

    1. Remove the skin from the garlic, and chop coarsely.
    2. Remove all the seeds and stalk from the pepper, and chop coarsely.
    3. Add all the ingredients into a food processor.
    4. Process all the ingredients into a paste.
      • If you want a coarse "chunky" hummus, process it for a short time.
      • If you want a smooth hummus, process it for a longer time.

    Emphasis and importance

    We would want to mark certain words as important or something different. HTML provides the <em> and <strong> tags.

    Emphasis

    We use the <em> tag to mark up such instance .

    Browser will render this as emphasis text as an italic text but it shoudn't be used to get italic text. To get italic we can use <i> tag or by adding css.Exmaple of emphasis

    I am glad you weren't late.

    Strong importance

    To emphasize important word we can bold them using <strong> tag, for example

    • This liquid is highly toxic.
    • I am counting on you. Do not be late!

    We can nest strong and emphasis inside one another if desired:

    This liquid is highly toxic — if you drink it, you may die.
