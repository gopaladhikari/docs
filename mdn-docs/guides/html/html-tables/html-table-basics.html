    This article will help in very basics of table such as rows, cells, heading etc.

    What is a table?

    A table is the structured set of data made up of rows and cells ( tablular data ) that indicate some kind of connection between data, for example person and their age or gender.

    How does table works ?

    The point of a table is that it is rigid. Information is easily interpreted by making visual associations between row and column headers. When HTML tables are implemented correctly, screen readers can interpreted in easily.

    Table styling

    For the table to be more effected on the web, CSS styling should be provied.

    When should you NOT use HTML tables?

    HTML table should be used for tabular data, this is what they are designed for. A lot of people use HTML table to layout web pages, for example one for header, footer etc. This is common before CSS support across multiple browser.

    Creating first table

    For creating first table, we need table and td elements. ts stands for table data.

    Hi, I'm your first cell. I'm your second cell. I'm your third cell. I'm your fourth cell.

    These cells are not placed underneath each other. td create a single cells which creates the first row and for every cell we add it grows longer.

    To stop this td growing longer and starts a new row we need tr element. tr stands for table row.

    Hi, I'm your first cell. I'm your second cell. I'm your third cell. I'm your fourth cell.
    Second row, first cell. Cell 2. Cell 3. Cell 4.

    Adding headers with elements

    To give the meaning of the table we need header. It goes in the start of the row or column.
