    Adding a caption to your table with caption

    We can give the caption to the table by putting the caption inside the table.

    Dinosaurs in the Jurassic period
    T-Rex Vulture Anaconda Meg Titnoboa

    Caption is brief description of the table. A caption is placed directly beneath the table tag.
